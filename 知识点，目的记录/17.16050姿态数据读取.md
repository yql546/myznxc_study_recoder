4.
输入输出模式配置，修改为pb8,9的


查看手册，进行寄存器操作，更改引脚模式的步骤及理解：


2. 第一步：GPIOB->CRH &= 0XFFFFFF0F;（清空PB9的旧配置）
0XFFFFFF0F 写成二进制，除了 bit7~bit4 是 0000，其他所有位全是 1。

按位与（&=） 的作用是：1不变，0清零。

所以这行代码的效果是：只把 bit7~bit4 变成 0000（即把 PB9 原来的 CNF 和 MODE 全部擦除），而 PB8、PB10~PB15 对应的 bit 因为跟 1 与，原封不动地保留下来。

3. 第二步：设置输入 |= 8 << 4（把 PB9 设为输入）
8 的二进制是 1000。

8 << 4 意思是左移 4 位，得到 1000 0000（即 bit7=1, bit6=0, bit5=0, bit4=0）。

按位或（|=） 的作用是：1 强制置 1，0 保持不变。

此时写入 bit7~bit4 的值为 1000：

bit7:6（CNF）= 10 → 代表 上拉/下拉输入模式（输入）。

bit5:4（MODE）= 00 → 代表 输入模式（复位状态）。

结论：PB9 变成输入模式。

4. 第三步：设置输出 |= 3 << 4（把 PB9 设为输出）
3 的二进制是 0011。

3 << 4 得到 0011 0000（即 bit7=0, bit6=0, bit5=1, bit4=1）。

写入 bit7~bit4 的值为 0011：

bit7:6（CNF）= 00 → 代表 通用推挽输出模式（输出）。

bit5:4（MODE）= 11 → 代表 输出模式，最大速度 50MHz。

结论：PB9 变成推挽输出模式。
